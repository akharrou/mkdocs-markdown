# MKDW ➤ Footnotes

## Synopsis

Footnotes are supported with the `[^1]` ... `[^1]: footnote content` syntax.

## Examples

=== "Adding footnotes"

    In
    : <space>

        ```
        Lorem ipsum.[^1] More text incoming.[^2]

        ...

        [^1]: first footnote content
        [^2]: second footnote content ... <br> <br> more content
        ```

    ---

    Out
    : <space>

        Lorem ipsum.[^1] More text incoming.[^2]
